## 简介
本系统基于梅花易数原理，提供完整的体用分析能力。系统通过时间起卦、两数法、三数法和手动选卦四种方式生成本卦、变卦和对卦，自动识别体用位置（上卦/下卦），计算体用元素的五行生克关系，并结合月令旺衰进行能量状态校准，最终输出三阶段（缘起-过程-终局）的综合判断。

### 体用关系判断算法
体用关系判断是系统的核心算法，通过比较体卦和用卦的五行属性来确定关系类型。

```mermaid
flowchart TD
Start(["开始"]) --> CheckSame{"体用是否相同？"}
CheckSame --> |是| ReturnHarmony["返回：体用比和"]
CheckSame --> |否| CheckGenerate{"用生体？<br/>用的生成=体"}
CheckGenerate --> |是| ReturnBenefit["返回：用生体吉"]
CheckGenerate --> |否| CheckCreate{"体生用？<br/>体的生成=用"}
CheckCreate --> |是| ReturnDisperse["返回：体生用泄"]
CheckCreate --> |否| CheckOvercome{"体克用？<br/>体的克制=用"}
CheckOvercome --> |是| ReturnSmallGood["返回：体克用小吉"]
CheckOvercome --> |否| CheckBeOvercome{"用克体？<br/>用的克制=体"}
CheckBeOvercome --> |是| ReturnBad["返回：用克体凶"]
CheckBeOvercome --> |否| ReturnFlat["返回：平"]
ReturnHarmony --> End(["结束"])
ReturnBenefit --> End
ReturnDisperse --> End
ReturnSmallGood --> End
ReturnBad --> End
ReturnFlat --> End
```

体用关系的五种类型及其判定标准：
1. **体用比和**：当体卦元素与用卦元素完全相同时
2. **用生体（吉）**：用卦元素的"生成"属性等于体卦元素
3. **体生用（泄）**：体卦元素的"生成"属性等于用卦元素  
4. **体克用（小吉）**：体卦元素的"克制"属性等于用卦元素
5. **用克体（凶）**：用卦元素的"克制"属性等于体卦元素

### 月令旺衰校准机制
月令旺衰校准通过干支历计算当前节气对应的地支和五行元素，然后使用能量状态函数进行校准。

```mermaid

GC-->>DE : 返回{branch, element, jieQi}
DE->>BD : getEnergyState(元素, 月令元素)
BD-->>DE : 返回能量状态(旺/相/休/囚/死)
DE->>DE : 计算原始/变卦/对卦的能量状态
```

月令元素对能量状态的影响：
- **相生关系**：月令元素生成该元素时，状态为"相"
- **被生关系**：该元素生成月令元素时，状态为"休"  
- **克制关系**：该元素克制月令元素时，状态为"囚"
- **被克关系**：月令元素克制该元素时，状态为"死"
- **同元素**：状态为"旺"

## 性能考虑
系统在性能方面的优化策略：
- 使用取余运算符避免复杂的条件判断
- 缓存干支历计算结果，避免重复计算
- 函数式编程风格，减少状态变更
- 模块化设计，便于测试和维护
