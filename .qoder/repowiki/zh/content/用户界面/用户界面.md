## 简介
本项目为"梅花义理"AI断卦系统，围绕用户界面（UI）构建了模块化的视图层，涵盖聊天界面、六十四卦展示界面、历史记录界面以及模态框系统。UI采用响应式设计，适配桌面与移动端；通过主题系统实现浅色/深色模式切换；通过统一的状态中心协调各组件间的数据与行为。本文档将深入解析UI组件架构、数据流与交互流程，并提供组件API参考、样式定制指南与最佳实践。

**更新** 新增divination-warning警告系统与ritual-guide引导系统的输入验证功能增强，改进了智能警告显示策略，增加了重复字符检测和更全面的模式识别能力，提升了用户体验的一致性和沉浸感。

## 架构总览
UI采用"入口编排 + 视图模块 + 工具与状态"的分层设计：
- 入口负责初始化、事件绑定、主题与抽屉控制、导出与反馈等全局逻辑
- 视图模块负责各自领域的渲染与交互
- 工具模块提供DOM与文本格式化能力
- 状态模块集中管理用户、历史、模型选择等共享数据
- **新增** 警告与引导系统集成到增强的输入状态管理中

### 增强输入验证系统（isMeaningfulDivinationQuestion）
职责
- 识别有意义但未被解析的占卜问题
- 检测重复字符模式（如"对对对"、"啊啊啊"、"111"）
- 拦截纯寒暄与测试短语
- 支持纯符号、纯数字和过短内容检测
- 提供全面的模式识别能力

关键特性
- 基于 isMeaningfulDivinationQuestion 判定逻辑的增强版本
- 集成重复字符检测算法（isRepeatedChars）
- 支持多种语言和标点符号的标准化处理
- 与 DivinationEngine.parseFromText 配合使用
- 通过 handleTextInputChange 动态控制显示/隐藏

```mermaid
flowchart TD
Input["用户输入"] --> Normalize["标准化处理<br/>清除标点符号和空格"]
Normalize --> CheckEmpty{"是否为空?"}
CheckEmpty --> |是| ReturnFalse["返回false"]
CheckEmpty --> |否| CheckPatterns{"匹配寒暄/测试模式?"}
CheckPatterns --> |是| ReturnFalse
CheckPatterns --> |否| CheckRepeats{"检测重复字符?"}
CheckRepeats --> |是| ReturnFalse
CheckRepeats --> |否| CheckLength{"长度>=5?"}
CheckLength --> |否| ReturnFalse
CheckLength --> |是| CheckIndicators{"包含问题指示词?"}
CheckIndicators --> |是| ReturnTrue["返回true"]
CheckIndicators --> |否| CheckLong{"长度>=8?"}
CheckLong --> |是| ReturnTrue
CheckLong --> |否| ReturnFalse
ReturnTrue --> ParseCheck{"已解析到卦象?"}
ParseCheck --> |是| ReturnTrue
ParseCheck --> |否| ReturnFalse
```

## 故障排查指南
- 模态框无法关闭
  - 检查 initModals 是否已调用
  - 确认点击背景或关闭按钮是否触发
- 滑动历史项无效
  - 确认 pointer 事件是否被垂直滚动锁定
  - 检查滑动阈值与 transform 设置
- 聊天消息不显示
  - 确认 addMessage 调用与容器存在
  - 检查 formatMarkdown 输出是否为空
- 动爻标记不显示
  - 确认传入 movingYao 与原始卦数据一致
  - 检查 CSS 类 .is-moving 是否生效
- 主题切换异常
  - 检查本地存储键值与 data-theme 设置
  - 确认 [data-theme="dark"] 样式覆盖是否正确
- **新增** 起卦警示不显示
  - 检查 handleTextInputChange 是否正确调用
  - 确认 isMeaningfulDivinationQuestion 判定逻辑
  - 检查 divination-warning 元素是否存在且未被其他样式覆盖
  - **新增** 检查用户引导状态：localStorage 中的 meihua_has_seen_qigua_guide 键值
- **新增** 净心引导不出现
  - 确认 handleTextInputBlur 是否在键盘确认时触发
  - 检查 btn-time-divine 是否正确添加 breathing 类
  - 确认 CSS 动画是否正常加载
  - **新增** 检查输入验证结果：确保 isMeaningfulDivinationQuestion 返回 true 且未解析到卦象
- **新增** 重复字符检测失效
  - 检查 isRepeatedChars 函数是否被正确调用
  - 确认标准化处理是否正确移除了标点符号和空格
  - 检查正则表达式模式是否正确匹配重复字符
